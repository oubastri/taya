# To All You Athletes

Workout tracker for athletes. Built with Next.js 14, TypeScript, Tailwind CSS, and App Router.

## Setup

1. Install Node.js (v18+) if you haven’t already.
2. From this folder, run:

```bash
npm install
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser (or on your phone on the same Wi‑Fi for a mobile feel).

## Features

- **Stats view**: Total moves (workouts) with breakdown (total, this year, month, week), rhythm score, and streak visualization.
- **Calendar view**: Month calendar with black dots on days you logged a workout; tap a day to see or add a description.
- **Add/Edit workout**: Tap the green + button or Edit on a day to log or update a workout (e.g. “Crossfit at 7:30am. Bench press, jump rope, burpees.”).
- Data is stored in your browser (localStorage); no backend.
